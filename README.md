# Ābolu ievārijuma kalkulators

## Apraksts:
Ši programma:
-Atvieglo ābola ievārījuma izstrādāšanas procesu
-Aprēķina cik daudz cukuru (kg) ir nepieciešams pēc lietotāja ievadītam ābolu svaram
-Aprēķina kopējas izmaksas, ņemot vērā cukura cenu (par 1 kg), kas lietotājs ievada
-Aprēķina gala ievārījuma masu (kg), ja ir zināms kā vārīšanas laikā 10% no kopējās masas iztvaiko

## Kā palaist programmu:
1. Lejupielādē failu iev_kalkulators.py
2. Palaid Python vidē
3. Ievadi prasītos datus
